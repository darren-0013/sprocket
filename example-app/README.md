## Setup
The example-app's frontend uses yarn,
installation instructions can be found here:
https://yarnpkg.com/en/docs/install

```bash
# Start the Scheme backend:
$ scheme < api.scm
# Then in a new terminal, start the frontend:
$ yarn
$ yarn start
```
